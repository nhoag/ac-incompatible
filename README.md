# Acqinc

Build a simple list of incompatible modules from Acquia Cloud documentation:

* https://docs.acquia.com/articles/module-incompatibilities-acquia-cloud

## Installation

```
$ git clone git@github.com:nhoag/acq-incompatible.git
$ cd acq-incompatible
$ bundle install
```

## Usage

```
$ ./bin/run
```
